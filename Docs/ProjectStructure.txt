Interface Navigation Structure:
1. Login Page (Default)
   └── Redirects to respective dashboard based on role

2. Customer Dashboard
   ├── Book Flight
   ├── Order Food
   └── View History

3. Admin Dashboard
   ├── User Management
   └── View System Data

4. Operations Dashboard
   ├── Flight Management
   └── Crew Management

5. Analyst Dashboard
   ├── Flight Data
   ├── Passenger Data
   └── Food Sales Data

6. Check-in Dashboard
   ├── Search Passenger
   └── Register Luggage
